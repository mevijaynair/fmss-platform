❓ TROUBLESHOOTING

Ports in use?
   lsof -i :8000    # Find process
   kill -9 <PID>    # Kill it

Services won't start?
   • Check Node.js: node --version
   • Check npm: npm --version
   • View terminal output for errors
   • Try manual startup instead of scripts

Database issues?
   • Delete .db files: rm data/*.db*
   • Services recreate them on startup
   • Demo data automatically seeded

Subdomain not routing?
   • Development: Use direct ports (localhost:3000, etc)
   • Check /etc/hosts (Windows/Mac/Linux)
   • Production: Verify DNS and nginx config

See README.md and GETTING_STARTED.md for more troubleshooting.
