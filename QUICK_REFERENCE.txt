✅ STEP-BY-STEP CHECKLIST

WEEK 1:

[ ] Step 1: Go to Namecheap.com
    • Search for: fmss.ae
    • Add to cart
    • Checkout with credit card
    • Cost: ~$15
    • Time: 5 minutes
    ✓ DONE: You own fmss.ae

[ ] Step 2: Go to DigitalOcean.com
    • Click "Sign Up"
    • Create account
    • Add credit card (get $200 free credits!)
    • Cost: FREE (using credits)
    • Time: 10 minutes
    ✓ DONE: Account created

[ ] Step 3: Push to GitHub
    • Go to GitHub.com
    • Create account (if needed)
    • Create repo: "fmss-platform"
    • On your PC:
      cd H:\Football_ai\fmss-platform
      git init
      git add .
      git commit -m "Initial"
      git remote add origin [your-github-url]
      git push -u origin main
    • Cost: FREE
    • Time: 10 minutes
    ✓ DONE: Code in GitHub

[ ] Step 4: Deploy to DigitalOcean
    • In DigitalOcean: Create → App
    • Select: GitHub source
    • Pick: fmss-platform repo
    • Select: docker-compose.yml
    • Add Environment Variables:
      NODE_ENV=production
      DOMAIN=fmss.ae
      SAMS_JWT_SECRET=[generate-random]
      PORT=8000
      SAMS_PORT=3000
      CONTRACTS_PORT=3100
    • Click: DEPLOY
    • Wait: 3-5 minutes
    • Cost: FREE (using credits)
    • Time: 15 minutes
    ✓ DONE: Deployed!

[ ] Step 5: Update DNS
    • In Namecheap (your domain registrar)
    • Go to: Domain settings
    • Update Nameservers to DigitalOcean:
      ns1.digitalocean.com
      ns2.digitalocean.com
      ns3.digitalocean.com
    • Or: Create DNS records in DigitalOcean console
    • Cost: FREE
    • Time: 5 minutes
    • WAIT: 24-48 hours for propagation ⏳
    ✓ DONE: DNS updated

[ ] Step 6: Wait for DNS (24-48 hours)
    • Check every few hours:

    • Should show: Academy system login

    • Should show: Contracts dashboard

🆘 IF SOMETHING GOES WRONG

"fmss.ae doesn't load"
  → DNS hasn't propagated yet (wait 24-48 hours)
  → Check: https://mxtoolbox.com/
  → Or domain not pointing to DigitalOcean IP

"Shows error about certificate"
  → Usually auto-fixes within 5 minutes
  → Wait and refresh browser
  → Check DigitalOcean console

"Services not responding"
  → Check DigitalOcean app logs
  → Restart app in console
  → Make sure environment variables set correctly

"Can't login to SAMS/Contracts"
  → Check password (case-sensitive)
  → May need to reset in database
  → Contact support with error message
